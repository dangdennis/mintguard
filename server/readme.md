# Server

## setup and development

1. `npm install`
2. 3 terminal windows:
    1. `npm run dev` in one terminal to build app in watch mode
    2. `npm start` to start web server
    3. `npm run tsc:dev` to run tsc in watch mode

## production

deploy to fly.io
1. `npm run build`
1. dennis runs `fly deploy`
